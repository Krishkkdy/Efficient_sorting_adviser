Information


Bubble Sort

Compares 2 Elements and swaps it, If Previous is Smaller


Best Time Complexity: O(n)

Average Time Complexity: O(n)

Worst Time Complexity: O(n^2)

Selection Sort

Selects the Minimum element and swaps it with the first.


Best Time Complexity: O(n^2)

Average Time Complexity: O(n^2)

Worst Time Complexity: O(n^2)

Insertion Sort

The Array is virtually split into a sorted and an unsorted part. Values from the unsorted part are picked and placed in the sorted part.


Best Time Complexity: O(n)

Average Time Complexity: O(n^2)

Worst Time Complexity: O(n^2)

Merge Sort

Divides the array and sorts using Divide & Conquer Strategy.


Best Time Complexity: O(n log n)

Average Time Complexity: O(n log n)

Worst Time Complexity: O(n^2)

Quick Sort

Divides the array into smaller sub-arrays and then recursively sorts these sub-arraysusing 'pivot' element.


Best Time Complexity: O(n log n)

Average Time Complexity: O(n log n)

Worst Time Complexity: O(n^2)

Heap Sort

Heap Sort works by first transforming the input array into a max-heap or a min-heap them and recursively sorts it.


Best Time Complexity: O(n log n)

Average Time Complexity: O(n log n)

Worst Time Complexity: O(n log n)

Shell Sort

Shell Sort works by sorting pairs of elements far apart from each other, then progressively reducing the gap between elements to be compared and swapped.


Best Time Complexity: O(n log n)

Worst Time Complexity: O(n^2)

Counting Sort

Counting Sort works by counting the number of occurrences of each unique element in the input array and then placing them in sorted order.


Best Time Complexity: O(n + k)

Average Time Complexity: O(n + k)

Worst Time Complexity: O(n + k)

Radix Sort

Radix Sort is a non-comparative integer sorting algorithm that sorts elements by processing individual digits of numbers.


Best Time Complexity: O(nk) (k: digits)

Average Time Complexity: O(nk)

Worst Time Complexity: O(nk)

Tim Sort

Tim Sort works by dividing the input array into small blocks, sorting each block using insertion sort,merging that blocks using a modified merge sort.


Best Time Complexity: O(n log n)

Average Time Complexity: O(n log n)

Worst Time Complexity: O(n log n)
